## Használat

### Adatbázis:

Root könvtárból:

```bash
db/start.bat
```

### Backend:

```bash
npm start
```

Fejlesztéshez:

```bash
nodemon
```

## Telepítés

### Adatbázis telepítése:

Root könyvtárból

```bash
db/import.bat
```

### Backend:

Node csomagok telepítése:

```bash
npm i
```

TypeScript fájlok átkonvertálása JavaScript fájlokká:

```bash
npm run compile
```
